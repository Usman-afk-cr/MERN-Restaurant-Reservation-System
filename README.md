# MERN Restaurant Reservation System

## Project Overview

This project is a full-stack Restaurant Reservation System developed using MERN Stack (MongoDB, Express.js, React.js, Node.js). The system allows users to register, log in, create reservations, update reservations, and manage bookings. An admin dashboard is implemented to manage users, reservations, and menu items.

Advanced MongoDB concepts such as indexing, aggregation pipelines, query optimization, transactions, views, and analytics are integrated to enhance database performance and scalability.

---

## Features

### User Features

- User Registration
- User Login/Authentication
- JWT Authorization
- Make Reservation
- Edit Reservation
- Delete Reservation
- View Personal Reservations

### Admin Features

- View All Users
- Delete Users
- View Reservations
- Delete Reservations
- Add Menu Items
- Delete Menu Items

### MongoDB Features

- CRUD Operations
- Indexing
- Aggregation

- Query Optimization
- Transactions
- MongoDB Views
